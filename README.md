# Info

## Tech Stack

- pnpm
- Svelte + SvelteKit
- TypeScript
- Prettier + ESLint
- Tailwind CSS
- unplugin-icons + Iconify + Lucide
- GitHub Pages + GitHub Actions

## Commands

#### Initial Setup

pnpx sv create website

#### Install Packages

pnpm install

#### Update Packages

pnpm update

#### Run Development Server

pnpm dev

#### Build Local

pnpm build

#### Run Local Build

pnpm preview
